# 重新搭建我的个人博客

这是这个个人博客的第一篇示例文章。

我希望它不仅是一个展示页面，也是一套可以长期维护的写作系统：文章放在 `content/essays` 目录中，项目放在 `content/projects/projects.json` 中，个人介绍放在 `content/about.mdx` 中。

## 为什么要这样组织？

本地内容文件有几个优点：

- 写作体验简单直接；
- 适合版本管理；
- 构建时生成静态页面，访问速度快；
- 后续迁移到 CMS 或数据库时，也有清晰的数据边界。

## 示例代码块

```ts
export function sayHello(name: string) {
  return `你好，${name}`;
}
```
